FDRAnalysis

This application enables an analysis of false discovery rates (FDR) of mass spectrometry identification results. Currently it can accept Mascot, X!Tandem and OMSSA results and can analyse all of the searches separately but can also combine the searches from different search engines, enhancing confidence in the results.

Please use the menu buttons above to run the analysis.
